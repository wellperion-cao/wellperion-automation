# brand

웰페리온 (Wellperion) 브랜드 디자인 SSOT. 모든 콘텐츠는 본 자산 기준.


## 폴더

로고          브랜드 로고 (W 마크 포함 변형)
컬러          사업부별 컬러칩 (P.T·골프·스쿼시·체조·필라테스·G.X·파트너팀)
가이드        브랜드 가이드라인 (폰트·여백·톤매너)
시설 사진     시설 원본 사진 (현대하이페리온 한남)


## 표기 규칙 (필수)

변경 시 GM 결재. 브랜드 카테고리는 자율 진행 금지.

콘텐츠에서 건물명 "현대하이페리온" 사용 금지. "웰페리온"으로 통일.

"피트니스" 단어 금지. "스포츠클럽"으로 통일.

"동네 잡담"·"이웃" 톤 금지. 격조 톤만 허용 (특히 동부이촌동 카페).
