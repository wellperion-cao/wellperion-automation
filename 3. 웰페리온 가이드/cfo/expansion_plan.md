작성일: 2026-05-28
실무진: 나우열M
북극성 KPI: 월 OCF 흑자 유지 / 부서별 예산 집행률 ≤100%

## 현황

현재 CFO 라인: 지출현황 1페이지만 운영 중
목표: 재무 본업 4개 영역을 단계적으로 페이지로 신설하여 재무 가시성 확보

---

## 1. 매출현황 (Sales Dashboard)

### 목적
월별·채널별 매출을 한눈에 파악해 OCF 흑자 여부를 실시간 확인한다.

### 주요 섹션
1. 월별 매출 추이 차트 (바 그래프)
2. 채널별 매출 분해 (멤버십·PT·기타)
3. 이번 달 목표 달성률 게이지
4. 전월 대비 증감 요약 카드
5. 미수금·정산 대기 현황

### 데이터 소스
- Apps Script 연동 매출 집계 시트 (구글 스프레드시트)
- CRM 결제 데이터 (추후 측정)
- 웰페리온 회원 결제 내역 DB

### 북극성 KPI 연결
- 월 OCF 흑자 직결: 매출 합계가 영업현금흐름의 분자
- 목표 달성률 ≥100% 시 녹색, 미달 시 적색 표시

### 우선순위 및 일정
- 우선순위: 1순위 (지출현황과 쌍을 이루는 가장 핵심 지표)
- Q3 2026: v0.1 샘플 신설 (완료) → 데이터 소스 연동 착수
- Q4 2026: Apps Script 자동 갱신 연동 완료, v1.0 정식 운영

---

## 2. 예산집행 (Budget Execution)

### 목적
부서별·월별 예산 대비 실집행 현황을 추적해 집행률이 100%를 초과하지 않도록 관리한다.

### 주요 섹션
1. 부서별 예산 vs 실집행 비교 바 차트
2. 집행률 테이블 (부서명 / 예산 / 집행액 / 잔여 / 집행률%)
3. 초과 위험 경보 섹션 (집행률 90% 이상 자동 강조)
4. 월별 누적 집행 추이
5. 항목별 집행 상세 드릴다운

### 데이터 소스
- 지출현황.html Apps Script 시트 (카테고리별 집계 재활용)
- 예산 마스터 시트 (별도 구축 필요 — 나우열M 작성)
- Notion 업무자동화 DB 연동 가능성 검토

### 북극성 KPI 연결
- 부서별 예산 집행률 ≤100% 직결
- 집행률 초과 시 AI CFO → CEO 자동 경보 트리거 예정

### 우선순위 및 일정
- 우선순위: 2순위
- Q3 2026: 예산 마스터 시트 구축 + 페이지 v0.1 신설
- Q4 2026: 경보 자동화 연동, v1.0 정식 운영

## 3. 세무 (Tax & Compliance)

### 목적
부가세·원천세·4대보험 등 세무 납부 일정과 이행 현황을 체계적으로 관리한다.

### 주요 섹션
1. 연간 세무 캘린더 (납부일 30일 전 경보)
2. 부가세 신고 현황 (1기·2기 예정·확정)
3. 원천세 월별 납부 이행 체크리스트
4. 4대보험 납부 현황
5. 세무 이슈 / 리스크 메모 섹션

### 데이터 소스
- 국세청 전자신고 일정 (수동 입력 기준)
- 지출현황 시트 부가세 항목 연계
- 나우열M 직접 관리 항목 (세무사 연락처·신고 내역)

### 북극성 KPI 연결
- 세무 이행 불이행 시 OCF에 가산세 리스크 발생
- 납부 완료율 100% 유지가 재무 건전성 지표

### 우선순위 및 일정
- 우선순위: 3순위
- Q4 2026: 세무 캘린더 + 체크리스트 v0.1 신설
- Q1 2027: 세무사 API 연동 검토, v1.0 정식 운영

---

## 4. 결제관리 (Payment Management)

### 목적
법인카드·계좌이체·POS 등 결제수단별 현황과 미결제 건을 통합 관리한다.

### 주요 섹션
1. 결제수단별 월별 사용 현황 요약
2. 미결제·정산 대기 항목 목록
3. 법인카드 한도 사용률 모니터링
4. 정기 결제(구독) 관리 목록 (갱신일·금액·담당자)
5. 결제 승인 이력 및 반려 사유 로그

### 데이터 소스
- 지출현황 Apps Script 시트 (결제수단 필드 재활용)
- 법인카드사 웹 데이터 (수동 입력 기준)
- 정기 결제 마스터 목록 (나우열M 작성)

### 북극성 KPI 연결
- 미결제 건 0 유지 → 현금흐름 왜곡 방지
- 구독 낭비 항목 발굴 → 비용 최적화 → OCF 개선

### 우선순위 및 일정
- 우선순위: 4순위
- Q4 2026: 정기 결제 목록 + 결제수단 현황 v0.1 신설
- Q1 2027: 법인카드 연동 검토, v1.0 정식 운영
