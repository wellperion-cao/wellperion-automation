# CMO 라인 확장 기획서 v1.0

작성일: 2026-05-28
담당: AI CMO
실무진(최종 책임자): 김남욱 GM
버전: v1.0

---

## 개요

마케팅 핵심 병목은 노출 부족이 아니라 **노출→문의 전환 설계 부재**다.
현재 CMO 파이프라인(가공→검수→발행)은 콘텐츠 생산에 집중되어 있으며,
전환 경로 추적·SEO 구조화·캠페인 운영·CTA 단일화가 통합된 체계가 없다.

본 기획서는 CMO 라인을 4개 페이지로 확장해 전환 중심 마케팅 운영 체계를 수립한다.

---

## 북극성 KPI

- **채널별 문의 전환율** (노출 대비 litt.ly/wellperion 클릭 → 문의 완료 비율)
- 단순 노출 수·팔로워 수 단독 보고 금지. 반드시 전환율과 병기.

---

## 신설 4개 페이지 청사진

---

### 1. 캠페인 운영 (Campaign Operations)

**목적**: 채널별 콘텐츠 캠페인을 기획·실행·성과 측정하는 운영 허브

**주요 섹션**
1. 캠페인 캘린더 — 월별 주제·채널·발행 일정 매핑
2. 채널별 실행 현황 — IG / 네이버 블로그 / 네이버 카페(동부이촌동) 3채널 상태
3. 콘텐츠 큐 — 가공중 / 검수중 / 발행완료 / 반려 상태 트래킹
4. 성과 요약 — 채널별 도달·클릭·전환 수치 (주간 스냅샷)
5. 다음 액션 — 미발행 건 우선순위·담당자 지시

**데이터 소스**
- 업로드 캘린더 DB (Notion)
- IG Insights (Meta Graph API)
- 네이버 블로그 통계 (수동 수집 → 구글 시트)
- 네이버 카페 방문 통계 (수동)

**북극성 KPI 연결**
- 캠페인 기간 중 litt.ly 클릭 수 변화율
- 채널별 게시물 1건당 평균 문의 전환 기여 수

**실무진**: 김남욱 GM

**우선순위 및 일정**
- 우선순위: 2순위 (전환Funnel 다음)
- 신설 목표: 2026-06-2주차
- v1.0: 캠페인 캘린더 + 채널 현황 섹션 먼저 신설
- v1.1: 성과 요약 자동화 연동 (구글 시트 → HTML)

---

### 2. SEO 허브 (네이버 블로그·웹사이트 SEO)

**목적**: 검색 유입 경로를 체계화해 잠재 회원의 자연 유입을 증대

**주요 섹션**
1. 키워드 맵 — 타깃 키워드 목록·검색량·현재 순위 (네이버 기준)
2. 블로그 포스팅 로그 — 발행 URL·키워드·노출·클릭·체류시간
3. 내부 링크 구조 — 블로그 내 litt.ly CTA 삽입 현황
4. 경쟁 분석 — 동종 스포츠클럽 블로그 포스팅 빈도·키워드 비교
5. SEO 액션 리스트 — 우선 개선 포스팅·키워드 타깃 목표

**데이터 소스**
- 네이버 서치어드바이저 (수동 수집)
- 네이버 블로그 통계
- 구글 시트 (키워드 트래킹 마스터)
- Litt.ly 유입 경로 분석

**북극성 KPI 연결**
- 블로그 → litt.ly 클릭 전환율
- 월간 자연 검색 유입 문의 건수

**실무진**: 김남욱 GM

**우선순위 및 일정**
- 우선순위: 3순위
- 신설 목표: 2026-06-3주차
- v1.0: 키워드 맵 + 블로그 포스팅 로그 섹션
- v1.1: 네이버 서치어드바이저 데이터 정기 수집 자동화

### 3. 전환 Funnel (노출→인지→관심→문의→가입)

**목적**: 마케팅 각 단계의 전환율을 측정해 병목 지점을 찾고 개선 우선순위를 결정

**주요 섹션**
1. Funnel 시각화 — 5단계(노출·인지·관심·문의·가입) 단계별 수치·전환율
2. 단계별 측정 지표 — 각 단계 KPI 정의 및 데이터 소스 명시
3. 이탈 원인 분석 — 단계별 주요 이탈 원인·가설·개선안
4. 채널별 Funnel 비교 — IG / 블로그 / 카페 채널별 전환 경로 차이
5. 개선 실험 로그 — A/B 테스트·CTA 변경·콘텐츠 수정 후 전환율 변화 기록

**데이터 소스**
- Litt.ly 클릭 추적 (CTA 단일 진입점)
- IG Insights (도달·프로필 방문)
- 네이버 블로그 통계 (방문·이탈률)
- 실제 문의 건수 (COO 이슈 응답 DB 또는 수동 집계)
- 가입 전환 건수 (CPO 회원 DB)

**북극성 KPI 연결**
- 전체 Funnel 문의 전환율 (노출 대비 문의 완료 %)
- 단계별 탈락률 (병목 지점 식별)

**실무진**: 김남욱 GM

**우선순위 및 일정**
- 우선순위: **1순위** (가장 중요 — 현재 핵심 병목 직접 해결)
- 샘플 페이지 v0.1: 2026-05-28 신설 (본 기획과 동시)
- v1.0: 실측 수치 입력 + 채널별 Funnel 비교 섹션 완성 (2026-06-1주차)
- v1.1: Litt.ly API 연동으로 자동 수치 갱신

### 4. Litt.ly 분석 (CTA 단일 진입점 추적)

**목적**: 모든 채널 CTA의 단일 진입점인 litt.ly/wellperion 클릭 데이터를 분석해 채널 효율을 측정

**주요 섹션**
1. 일별·주별·월별 클릭 추이 — litt.ly 전체 클릭 수 시계열
2. 채널별 유입 비중 — IG / 블로그 / 카페 / 기타 채널별 클릭 기여율
3. 클릭→문의 전환율 — litt.ly 클릭 후 실제 문의 완료까지 전환 수
4. 링크 버튼별 클릭 분포 — litt.ly 내 개별 버튼(예약·상담·안내) 클릭 비중
5. 월간 리포트 요약 — 전월 대비 변화·주요 발견·다음 달 실험 제안

**데이터 소스**
- Litt.ly 대시보드 (수동 수집 → 구글 시트)
- 채널별 UTM 파라미터 (추후 도입 시 자동화 가능)
- 실제 문의 건수 대조 (COO/CPO 연계)

**북극성 KPI 연결**
- litt.ly 월간 클릭 수 (채널별 분리)
- 클릭→문의 전환율 (litt.ly 기준 최종 전환)

**실무진**: 김남욱 GM

**우선순위 및 일정**
- 우선순위: 4순위 (데이터 축적 후 의미 있는 분석 가능)
- 신설 목표: 2026-07-1주차 (최소 1개월 클릭 데이터 축적 후)
- v1.0: 일별 클릭 추이 + 채널별 유입 비중 수동 입력 시트 연동
- v1.1: UTM 파라미터 도입 + 자동 집계
